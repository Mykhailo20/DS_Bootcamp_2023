Під час дослідження показів акселерометра та гіроскопа для різних видів фізичної активності було отримано два датасети:
1) Тренувальний (містить 63529 записів);
2) Тестовий (містить 12557 записів).

Обидва датасети містять покази акселерометра та гіроскопа для 5-ти різних видів фізичної активності:
1) Присідання (Squat);
2) Ходьба (Walk);
3) Легкий біг (Jogging);
4) Випади на кожну ногу по черзі, тобто один випад на ліву ногу, один на праву і т.д. (Leg land);
5) Переміщення центру маси тіла з однієї ноги на іншу (Lateral squat slide).

Під час запису даних смартфон знаходився у правій руці у горизонтальному положенні (вісь OZ акселерометра смартфона спрямована перпендикулярно до горизонталі).

Тренувальний датасет було отримано шляхом об'єднання декількох датасетів, записаних для кожної фізичної активності окремо в різний час (в різні години однієї доби, або в різні дні). Це можна спостерігати, розглянувши вміст стовпця 'timestamp' тренувального датасету.
Тестовий датасет було записано відразу: всі вправи виконувалися послідовно із невеликими проміжками відпочинку між ними.

Обидва датасети містять дев'ять стовпців:
1) timestamp (мс) - містить інформацію про дату і час здійснення замірів.
2) time (с) - стовпець, доданий вручну для зручності графічного відображення показів акселерометра та гіроскопа у часі. Містить час здійснення заміру відносно початку виконання вправ (позначка 0 с) так, ніби згадані раніше вправи виконувалися безперервно в часі (тобто, як і у випадку із тестовим датасетом,  були записані за один раз, а не складалися з декількох датасетів, записаних з інтервалом в декілька годин). Для того, щоб об’єднати в один два датасети, різниця в часі між замірами яких є значною (наприклад, заміри для першого датасету присідань робилися о 13:20, а для другого – о 20:18 того ж дня), проміжок часу між двома датасетами замінявся середнім значенням періоду одного заміру першого датасету.
3) accX (м/с^2) - містить результати вимірювань осі OX акселерометра, тобто величину лінійного прискорення смартфона по осі OX акселерометра.
4) accY (м/с^2) - містить результати вимірювань осі OY акселерометра, тобто величину лінійного прискорення смартфона по осі OY акселерометра.
5) accZ (м/с^2) - містить результати вимірювань осі OZ акселерометра, тобто величину лінійного прискорення смартфона по осі OZ акселерометра.
6) gyrX (рад/с) - містить результати вимірювань осі OX гіроскопа, тобто величину кутової швидкості смартфона по осі OX гіроскопа.
7) gyrY (рад/с) - містить результати вимірювань осі OY гіроскопа, тобто величину кутової швидкості смартфона по осі OY гіроскопа. 
8) gyrZ (рад/с) - містить результати вимірювань осі OZ гіроскопа, тобто величину кутової швидкості смартфона по осі OZ гіроскопа.
9) activity - один із п'яти класів фізичної активності або клас 'No activity' (використовується для позначення пауз між вправами).
